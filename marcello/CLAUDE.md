# restaurant-hp プロジェクト

## 概要
ホームページを持っていない飲食店向けに、静的HTMLサイトを制作・納品するプロジェクト。
実績作りを目的とし、まずはテンプレートを整備して案件受注を目指す。

## 技術スタック
- HTML / CSS
- Tailwind CSS（CDN利用）
- 公開：Vercel または GitHub Pages

## フォルダ構成
```
src/         テンプレートHTMLファイル
docs/        設計書・進捗管理
```

## 作業ルール
- テンプレートは src/ に格納する
- 案件ごとのカスタマイズはブランチを切って作業する
- 納品物は別リポジトリに切り出す
